# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-06
### Added
- Universal, zero-dependency Whatspie SDK
- Support for Node.js, Bun, Deno, and Edge environments
- CJS, ESM, and TypeScript definitions out of the box
- Built-in logging with PII/secret redaction
- Automatic retries with exponential backoff and rate-limit handling
- Webhook signature verification utility
